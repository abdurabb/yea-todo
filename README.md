# TaskFlow - TODO Application

## Overview
TaskFlow is a full-stack TODO application built with **Node.js, Express, MongoDB, React, and Tailwind CSS**. Users can create and manage tasks, set deadlines, organize tasks into groups, filter tasks, and track their status. The application uses **JWT authentication** to secure API endpoints.

---

## Features
- **User Authentication:** Sign up with name, email, and password; log in with email and password.
- **Task Management (CRUD):**
  - Add new tasks
  - Get all tasks
  - Update task status or details
  - Delete tasks
- **Task Filtering & Search:**
  - Filter tasks by group, priority, or status
  - Search tasks by keywords
- **Pagination:** View tasks with pagination support
- **Responsive UI:** Built with React and Tailwind CSS
- **Secure Authentication:** JWT-based for protected routes

---

## Tech Stack
- **Frontend:** React, Tailwind CSS, React Router
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
- **Other Tools:** Axios, React Query, Nodemon, dotenv
